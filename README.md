# Human Behavior Analysis via Birth Data: A Statistical Study on Marital Stability

## 📌 Project Overview
Does "Astrological Compatibility" have an empirical basis in human relationship outcomes? This project leverages a large-scale dataset of Mexican divorce records to investigate whether chronological birth data (Zodiac signs) shows any statistically significant correlation with marital stability. By applying quantitative research methods, we aim to bridge the gap between social myths and data-driven insights into human behavior.

## 🔬 Core Research Questions
- **Zodiac Distribution:** Is there a statistically significant prevalence of certain zodiac signs within divorce records?
- **Compatibility Myth Testing:** Do "incompatible" element pairs (e.g., Fire vs. Water) exhibit shorter marriage durations or higher frequencies of divorce?
- **Predictive Potential:** Can we identify patterns in marital outcomes based on birth-related features using statistical modeling?

## 🛠️ Tech Stack & Methodology
- **Language:** Python
- **Libraries:** Pandas, Matplotlib, Seaborn, Scipy
- **Approach:** Exploratory Data Analysis (EDA), Statistical Hypothesis Testing (Chi-square, T-test), Research Design
